# EV-Battery-Charging-Using-CC-CV-Method
## Overview
This project focuses on the design and simulation of an Electric Vehicle (EV) battery charging system using the Constant Current-Constant Voltage (CC-CV) charging technique in MATLAB/Simulink.

## Objectives
- Implement the CC-CV charging algorithm.
- Monitor battery voltage, current, and state of charge (SOC).
- Analyze charging characteristics during different charging phases.

## Features
- Constant Current (CC) charging phase
- Constant Voltage (CV) charging phase
- SOC-based charging control
- Voltage and current monitoring
- MATLAB/Simulink-based simulation

## Software Used
- MATLAB
- Simulink

## Working Principle
The battery is initially charged at a constant current until the battery voltage reaches a predefined threshold. The charging mode then transitions to constant voltage, where the charging current gradually decreases while maintaining the desired terminal voltage.

## Results
- Simulated battery charging behavior under CC-CV control.
- Observed voltage, current, and SOC variations during charging.
- Verified smooth transition between CC and CV charging phases.
